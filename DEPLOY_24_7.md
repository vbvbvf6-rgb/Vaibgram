Этот проект можно запускать на бесплатных платформах без банковской карты. Для этого репозитория самый простой вариант сейчас — `Replit`, потому что здесь уже есть `.replit`, а приложение стартует одной командой: `python main.py`.

## Рекомендуемый вариант: Replit

### Что уже готово в проекте
- Файл `.replit` уже добавлен
- Приложение слушает порт из переменной `PORT`
- Точка входа: `python main.py`

### Как запустить на Replit
1. Открой `https://replit.com`
2. Создай новый Repl из GitHub-репозитория
3. Импортируй этот проект
4. Убедись, что команда запуска — `python main.py`
5. Нажми `Run`

### Переменные окружения
Если используешь внешнюю базу данных, добавь в `Secrets`:

```env
DATABASE_URL=postgresql://user:password@host:5432/dbname
PORT=8000
```

Если внешняя база не используется, достаточно:

```env
PORT=8000
```

### Публичная ссылка
После запуска Replit выдаст публичный URL. Открывай приложение по адресу:

```text
https://<your-repl-domain>/static/index.html
```

## Важное ограничение

Для стабильных звонков и видеозвонков одного только хостинга мало. WebRTC лучше работает, когда:
- сайт открыт по `https`
- оба пользователя выдали доступ к микрофону/камере
- при сложной сети есть `TURN`-сервер

Сейчас проект использует публичные `STUN`-серверы. Этого часто хватает, но для части мобильных сетей и строгих NAT может понадобиться отдельный `TURN`.
